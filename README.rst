========
Rolldice
========

|TravisCI| |CratesIO|

Simple program that generates random dice rolls given a typical representation for
number of dice and number of sides.

For example:

- "2d10" will roll 2 dice with 10 sides each
- "4d6" will roll 4 dice with 6 sides each
- etc....
